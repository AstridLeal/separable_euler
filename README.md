# Separación de variables + Método de Euler

## Descripción
Este repositorio contiene un ejemplo sencillo que compara la solución analítica obtenida por separación de variables con una aproximación numérica usando el método explícito de Euler.

**Ecuación:** dy/dt = y(1 - y),  y(0) = 0.2  
**Intervalo:** t ∈ [0, 1]  
**Paso Euler:** h = 0.2
